    简述:关于DynamicWallDemo
        这个 Project 为了实现动态壁纸的功能.
        采用 FFmpeg 和 DirectX 9 实现的将视频文件解码并绘制到桌面.
        需要配置相应的环境才可以编译相关 Project.

        依赖:
            FFmpeg 动态链接库    ... 
            DirectX 9 动态链接库 ... d3d9.dll, d3dx9_43.dll.
